Movie Review Sentiment Prediction

Github Page

This script analyzes the sentiment (positive or negative) of a movie review based on the connotation of the words in the review. The script has a training phase and testing phase. With this information, the code predicts the sentiment of the reviews in the testing file. This project was done as a part of my Data Mining class (CS 484).
